# Infrastructure notes

Deze map is gereserveerd voor aanvullende infra-assets rond de nieuwe web-stack.

Voor deze foundation-branch staat de actieve containerconfiguratie in:

- `apps/web/Dockerfile`
- `docker-compose.yml`

De `infra/` map blijft beschikbaar voor latere uitbreidingen zoals deployment manifests, reverse-proxy config of observability-assets.
